# Frontend Setup Instructions

## Prerequisites
- Node.js (v14 or later)
- npm (Node Package Manager)

## Installation Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/Linuxnapratica/esp32-dht22-iot-monitor.git
   cd esp32-dht22-iot-monitor
   ```
2. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```
   The application should now be running at `http://localhost:3000`.

## Features
- Real-time temperature and humidity monitoring.
- Easy-to-use interface for displaying sensor data.
- Configuration settings for adjusting sensor thresholds.
- Responsive design for mobile and desktop views.
- Dashboard for quick insights into environmental data.
